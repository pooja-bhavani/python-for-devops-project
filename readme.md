# Internal DevOps Utilities API

## Aim

This project aims to deliver production grade Utilities and API's o internal teams:

- AWS resources
- Log Analysis
- Metrics

### Usage
```
git clone https://github.com/pooja-bhavani/python-for-devops-project.git
```

### Setup python environment
```
python3.14 -m venv venv
source venv/bin/activate
```

### Install requirements
```
pip install requirements.txt
```

### run applications
```
python main.py
```
